## 🚀 التشغيل

### المتطلبات
- Node.js 18+ (تم البناء والاختبار على Node 22)

### الخطوات

```bash
cd backend
npm install --ignore-scripts
node server.js
```

> **ملاحظة عن `--ignore-scripts`:** المشروع يستخدم `nedb` (قاعدة بيانات JS خالصة بدون أي اعتمادات native) عمداً، لضمان التشغيل في أي بيئة بدون أدوات بناء (`node-gyp`, Python, C++ compiler). كل الحزم الأخرى (`express`, `bcryptjs`, `jsonwebtoken`, `multer`, `cors`) خالصة JS أيضاً.

عند أول تشغيل، السيرفر يقوم تلقائياً بـ:
1. إنشاء قاعدة بيانات NeDB في `backend/data/`
2. زرع (Seed) بيانات أولية: 4 أوراد، 4 مقاطع صوتية، 3 فيديوهات، 8 أحزاب دلائل الخيرات
3. إنشاء حساب مدير افتراضي

### الوصول

| الرابط | الوصف |
|---|---|
| `http://localhost:3000/` | الموقع العام |
| `http://localhost:3000/admin/login.html` | لوحة التحكم |

### بيانات الدخول الافتراضية

| الحقل | القيمة |
|---|---|
| اسم المستخدم | `admin` |
| كلمة المرور | `admin123` |

> ⚠️ **مهم:** غيّر كلمة المرور فوراً من **لوحة التحكم → الإعدادات** قبل النشر.

## 🔜 التوسعات المستقبلية المقترحة

1. **الترقية لقاعدة بيانات production** (PostgreSQL/MongoDB) — الـ wrapper في `database.js` يسهّل ذلك.
2. **رفع الملفات لتخزين سحابي** (AWS S3 / Cloudflare R2) بدل التخزين المحلي.
3. **صفحات "عن الطريقة" و"تواصل معنا"** المرتبطة في الـ Footer (روابط جاهزة، الصفحات غير منشأة).
4. **بحث نصي** في الأوراد والصوتيات والدلائل.
5. **نظام تنبيهات** (Push Notifications) لورد اليوم.
